# Presentation on LeanChem Connect — Project Deliverables

Slide presentation covering platform modules, integrations, deliverables, and roadmap.

## Files

| File | Purpose |
|------|---------|
| `LeanChem_Connect_Project_Deliverables.html` | 16-slide presentation (browser + print) |
| `LeanChem_Connect_Project.pdf` | PDF export |
| `../APP_DEVELOPMENT_BLUEPRINT.md` | Full written blueprint |

## View in browser

Open `LeanChem_Connect_Project_Deliverables.html` in Chrome or Edge.

- **Arrow keys** / **Page Up/Down** — navigate slides
- **Save as PDF** button (or `Ctrl+P`) — export

When printing to PDF:

1. Destination: **Save as PDF**
2. Layout: **Landscape**
3. Enable **Background graphics**
4. Margins: **None** (if available)

## Generate PDF automatically

From repository root:

```bash
node scripts/generate_leanchem_presentation_pdf.mjs
```

Output: `docs/leanchem-connect-presentation/LeanChem_Connect_Project_Deliverables.pdf`
